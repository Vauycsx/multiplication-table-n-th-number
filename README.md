# multiplication-table-n-th-number
A simple Python program that greets the user and prints the multiplication table for a number entered by the user.

**Features**

*- Asks for your name and greets you.*

*- Prompts you to enter a number.*

*- Displays the multiplication table for numbers 1 to 10.*

*- Friendly output that references your name.*


**How to Run**

*- Make sure you have Python installed (version 3.x recommended).*

*- Clone or download this repository.*

*- Run the program.*

**Example**

```
Your name: Max
hello Max
enter an number: 5
5 * 1 = 5
5 * 2 = 10
5 * 3 = 15
5 * 4 = 20
5 * 5 = 25
5 * 6 = 30
5 * 7 = 35
5 * 8 = 40
5 * 9 = 45
5 * 10 = 50
Max this is your multiplication table for 5
```
